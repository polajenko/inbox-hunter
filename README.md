# InboxHunter

AI-powered lead generation and automation platform that helps you build email lists by automatically discovering and signing up for newsletters on landing pages.

## Download

Download the latest version for your operating system:

| Platform | Download |
|----------|----------|
| **Windows** | [Download .msi installer](https://github.com/polajenko/inbox-hunter/releases/latest) |
| **macOS (Apple Silicon)** | [Download .dmg (M1/M2/M3/M4)](https://github.com/polajenko/inbox-hunter/releases/latest) |
| **macOS (Intel)** | [Download .dmg (x64)](https://github.com/polajenko/inbox-hunter/releases/latest) |
| **Linux** | [Download .deb or .AppImage](https://github.com/polajenko/inbox-hunter/releases/latest) |

## Installation

### Windows

1. Download the `.msi` file from the [latest release](https://github.com/polajenko/inbox-hunter/releases/latest)
2. Double-click the installer to install
3. Run InboxHunter from the Start menu

### macOS

1. Download the `.dmg` file for your Mac:
   - **Apple Silicon** (M1/M2/M3/M4): `InboxHunter_*_aarch64.dmg`
   - **Intel**: `InboxHunter_*_x64.dmg`
2. Open the `.dmg` file
3. Drag InboxHunter to your Applications folder
4. On first launch, right-click the app and select "Open" (required for unsigned apps)

> **Note**: Since InboxHunter is not signed with an Apple Developer certificate, macOS will show a security warning. Right-click and select "Open" to bypass this, or go to System Preferences > Security & Privacy to allow the app.

### Linux (Debian/Ubuntu)

```bash
sudo dpkg -i inboxhunter_*.deb
```

### Linux (AppImage)

```bash
chmod +x InboxHunter_*.AppImage
./InboxHunter_*.AppImage
```

## Features

- **AI-Powered Form Detection** - Uses GPT-4o Vision to intelligently identify and fill signup forms
- **Automated Email List Building** - Automatically signs up for newsletters across multiple landing pages
- **Smart Data Generation** - Generates realistic contact information using AI
- **Progress Tracking** - Real-time monitoring of signup progress and success rates
- **Export Options** - Export your collected data in various formats

## Auto-Updates

InboxHunter automatically checks for updates and will notify you when a new version is available. Updates are downloaded and installed seamlessly in the background.

## System Requirements

- **Windows**: Windows 10 or later (64-bit)
- **macOS**: macOS 11 (Big Sur) or later (Apple Silicon or Intel)
- **Linux**: Ubuntu 20.04+ or equivalent (64-bit)
- **Internet connection** required for AI features
